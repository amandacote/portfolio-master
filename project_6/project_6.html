Visual Instructions Poster

This Project required us to pick a handheld object and visualize how to use that object correctly through a number of steps. We couldn't use words, it had to all be described through illustrations. I chose a Dead Blow Hammer and I went for a layout where it not only described how to use it, but also how not to use it. You have to be careful with this tool because you could easily damage it, so I thought it would make sense to bring attention to that.
